# frontend

> Next.js dApp for **[HeliQuant](https://github.com/HeliQuant)** trading firm marketplace on Mantle.
>
> Submitted to **Mantle Turing Test Hackathon 2026** — Track 1: AI Trading & Strategy.

## Pages

| Route | Purpose |
|---|---|
| `/` | Landing with hero metrics + 3-layer architecture + 6 deployed contracts |
| `/firms/heliquant` | HeliQuant firm detail — strategy roster + iteration history (V0 → V5) |
| `/jobs/new` | ERC-8183 escrow Job creation form (USDC deposit + duration + perf fee) |
| `/whales` | Live whale watchlist auto-detected from Mantle DEX on-chain activity |

## Quickstart

```bash
pnpm install
pnpm dev          # http://localhost:3000
pnpm build        # production build
pnpm start        # serve production build
```

## Configuration

Optional env vars (`.env.local`):

```bash
# Get from https://cloud.walletconnect.com (recommended for production)
NEXT_PUBLIC_WC_PROJECT_ID=your-walletconnect-project-id
```

If unset, RainbowKit falls back to a demo project id (fine for dev/demo, not for production).

## Stack

- **Next.js 16** (App Router, Server Components)
- **Tailwind v4** (utility-first styling)
- **RainbowKit + Wagmi v2** (wallet connection)
- **viem** (typed Ethereum library)
- **Mantle Sepolia + Mainnet** preconfigured

## Notable design

- Server components for SEO + static rendering of landing content
- Client islands (`"use client"`) only for wallet connect button
- `getDefaultConfig` from RainbowKit is client-only — kept in `src/lib/wagmi.ts` with `"use client"` directive
- Server-safe contract constants extracted to `src/lib/contracts.ts`
- Whale watchlist data read at request time from `public/data/whale_watchlist.json` (regenerated by [agents](https://github.com/HeliQuant/agents) `scripts/07_build_whale_watchlist.py`)

## Architecture

```
src/
├── app/
│   ├── layout.tsx              ← root layout + Providers
│   ├── page.tsx                ← landing
│   ├── providers.tsx           ← Wagmi + RainbowKit + ReactQuery
│   ├── firms/heliquant/page.tsx
│   ├── jobs/new/page.tsx
│   └── whales/page.tsx
├── components/
│   ├── header.tsx
│   └── connect-wallet.tsx
└── lib/
    ├── contracts.ts            ← server-safe deployed addresses
    └── wagmi.ts                ← client-only RainbowKit config

public/data/whale_watchlist.json  ← synced from agents repo
```
